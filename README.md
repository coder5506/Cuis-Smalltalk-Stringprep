# Cuis-Smalltalk-Stringprep

Stringprep and profiles for [Cuis Smalltalk](https://cuis.st).

## License

[MIT License](LICENSE)

## Features

- [stringprep](https://datatracker.ietf.org/doc/html/rfc3454)
- [Profiles](https://datatracker.ietf.org/doc/html/rfc3454#section-1)
  - [nameprep](https://datatracker.ietf.org/doc/html/rfc3491)
  - [nodeprep](https://xmpp.org/rfcs/rfc3920.html#nodeprep)
  - [resourceprep](https://xmpp.org/rfcs/rfc3920.html#resourceprep)
  - [saslprep](https://datatracker.ietf.org/doc/html/rfc4013)

## Dependencies

- [Cuis-Smalltalk-Regexp](https://github.com/coder5506/Cuis-Smalltalk-Regexp)
- [Cuis-Smalltalk-Unicode](https://github.com/coder5506/Cuis-Smalltalk-Unicode)
